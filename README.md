# Overview:
A CLI application that helps users track job applications, manage their progress, and stay organized during the job search process.
Built to solve the problem of losing track of applications, statuses, and deadlines across multiple companies.

## Current Build Status:
![Build Status](https://github.com/WhoThisPerson/job-application-tracker/actions/workflows/ci-build.yaml/badge.svg)

## Features:
- Create and manage existing job applications from Google Sheets
- Track application statuses
- Store company job portal information (if any)

## Tech Stack:
- Typescript
- Node.js

## High-Level Design:
- Stores job application and company details locally
- Pushes to/Pulls from connected Google Sheets document via its API

For more info, check out *DESIGN.md*
